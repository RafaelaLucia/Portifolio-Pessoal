Gerencia de Tempo

O LabWatch foi o Trabalho de Conclusão de Curso do Curso Técnico em ADS do SENAI de Informática. Realizado em Equipe, o site visava resolver o problema de uma empresa real: A LabWare - Uma empresa que possui um software de automação para laboratórios. O objetivo do LabWatch foi criar uma aplicação que resolvesse o problema interno da Empresa: O gerenciamento de Tempo. O Sistema foi feito em C# e .NET, com React e versão para celular com React Native.
